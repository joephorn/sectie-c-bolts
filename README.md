# Sectie-C Bolts
Design tool voor Sectie-C's logo.

[(demo)](https://joephorn.github.io/sectie-c-bolts/)

## Features
- Wisselen tussen logo configuraties met shortcuts
- Exporteren naar SVG / PNG sequence / WebM / MP4 / Color + Matte (voor WebM met transparante achtergrond - combineer deze via de command line)

## Built with
- JavaScript
- Paper.js
- GSAP

## Getting started
```bash
git clone https://github.com/joephorn/sectie-c-bolts
npx serve .
```
